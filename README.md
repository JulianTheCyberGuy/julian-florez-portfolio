<h1>Florez Digital Solutions – Portfolio</h1>

<p>
Professional portfolio site for <strong>Florez Digital Solutions</strong>, focused on web development and automation services.
</p>

<p>
This repository contains the source code for a responsive, lightweight, static portfolio website deployed via GitHub Pages.
</p>

<hr>

<h2>Overview</h2>

<p>This portfolio showcases:</p>

<ul>
    <li>Web application development</li>
    <li>Backend systems experience</li>
    <li>Python-based automation tools</li>
    <li>Clean architectural design principles</li>
    <li>Practical business-focused solutions</li>
</ul>

<p>
The site is intentionally built using <strong>vanilla HTML, CSS, and JavaScript</strong> to prioritize:
</p>

<ul>
    <li>Performance</li>
    <li>Maintainability</li>
    <li>Simplicity</li>
    <li>Zero framework dependency</li>
    <li>Fast load times</li>
</ul>

<hr>

<h2>Tech Stack</h2>

<ul>
    <li>HTML5 (semantic structure)</li>
    <li>CSS3 (custom responsive layout)</li>
    <li>Vanilla JavaScript (light interaction logic)</li>
    <li>GitHub Pages (static hosting)</li>
    <li>Git for version control</li>
</ul>

<p>
No frontend frameworks were used to maintain minimal overhead and ensure portability.
</p>

<hr>

<h2>Architecture Decisions</h2>

<h3>Why Static?</h3>

<p>This portfolio does not require:</p>

<ul>
    <li>Server-side rendering</li>
    <li>Database connectivity</li>
    <li>Authentication</li>
    <li>CMS overhead</li>
</ul>

<p>Using a static architecture ensures:</p>

<ul>
    <li>Near-zero attack surface</li>
    <li>High availability</li>
    <li>CDN-level performance via GitHub Pages</li>
    <li>Minimal operational complexity</li>
</ul>

<hr>

<h2>Project Structure</h2>

<pre>
julian-florez-portfolio/
│
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
├── assets/
│   ├── images/
│   └── icons/
└── README.md
</pre>

<h3>Separation of Concerns</h3>

<ul>
    <li><strong>index.html</strong> → semantic layout</li>
    <li><strong>css/styles.css</strong> → presentation layer</li>
    <li><strong>js/main.js</strong> → behavior layer</li>
    <li><strong>assets/</strong> → static resources</li>
</ul>

<hr>

<h2>Performance Considerations</h2>

<ul>
    <li>No third-party libraries</li>
    <li>No blocking JavaScript</li>
    <li>Deferred script loading</li>
    <li>Minimal DOM complexity</li>
    <li>Lightweight CSS variables for theming</li>
</ul>

<p>Target: fast first paint and low bundle size.</p>

<h2>Local Development</h2>

<p>Clone the repository:</p>

<pre>
git clone https://github.com/JulianTheCyberGuy/julian-florez-portfolio.git
cd julian-florez-portfolio
</pre>

<p>Run a local server:</p>

<pre>
python -m http.server 8000
</pre>

<p>Visit:</p>

<pre>
http://localhost:8000
</pre>
